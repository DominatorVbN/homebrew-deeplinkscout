# DominatorVbN Homebrew Tap

Homebrew tap for DominatorVbN's macOS applications.

## Installation

```bash
brew tap DominatorVbN/DominatorVbN
brew install deeplinkscout
```

## Available Formulae

- **deeplinkscout** - Test and manage deeplinks for iOS and macOS development

## Updating

```bash
brew update
brew upgrade deeplinkscout
```
